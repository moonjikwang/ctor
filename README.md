### 📝 프로젝트 개요

  #### 프로젝트&스터디 모집게시판
    - 목록페이지 조건필터기능
    - 프로젝트 참여기능, 모집인원 초과시 자동마감
    - 강사회원 참여시 멘토 태그 추가
   #### 익명 게시판
    - 형용사 + 생물 랜덤생성하여 닉네임 부여(중복방지처리)
    - CK에디터 적용 및 이미지 첨부 가능
  #### 마이페이지
    - 자기소개, Skill역량체크, 참가한 프로젝트 및 게시글 수 댓글 수 확인 및 프린트기능
  #### PWA 적용 및 웹 푸시 구현
    - PWA 적용하여 PC & 모바일에 앱으로 설치 가능
    - 내 게시글에 댓글 작성시 푸시알림 구현
 ####  API
    - 카카오 REST API 소셜로그인 구현
    - 프로젝트에서 API 제공. 소개페이지.
